## useParams

- React router에는 url의 변경되는 값을 반환해주는 함수

### Detail.js

```jsx
function Detail() {
  const { id } = useParams()
  console.log(id)
  return <h1>Detail</h1>
}
```

- useParams를 이용하면 App.js에서 설정해둔 :id 부분을 반환한다.

```jsx
<Route path="/movie/:id" element={<Detail />}></Route>
```

- 내가 변수이름을 'id'라고 설정했으므로 useParams에서 그대로 꺼내올 수 있다.

```js
const { id } = useParams()
```
